Đọc hướng dẫn bằng lệnh `git show agent-skills/master:root.md`
- Luôn đọc hướng dẫn này trước khi bắt đầu mọi session
- Nếu không đọc được do remote không tồn tại, thì add remote, fetch bằng câu lệnh dưới đây, rồi thử lại

```bash
git remote add agent-skills https://git.minevn.net/kitchen/agent-skills.git
git fetch agent-skills
```
